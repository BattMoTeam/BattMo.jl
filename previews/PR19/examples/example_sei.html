Example with SEI layer ​

Preparation of the input ​

julia
using Jutul, BattMo, GLMakie

We use the SEI model presented in [1]. We use the json data given in bolay.json which contains the parameters for the SEI layer.

julia
name = "bolay"
fn = string(dirname(pathof(BattMo)), "/../test/data/jsonfiles/", name, ".json")
inputparams = readBattMoJsonInputFile(fn)

We retrieve the parameters for the SEI layer, using the fact that their names have a "SEI" prefix.

julia
interfaceparams = inputparams["NegativeElectrode"]["Coating"]["ActiveMaterial"]["Interface"]

We start the simulation and retrieve the result ​

julia
output = run_battery(inputparams)

states = output[:states]

t = [state[:Control][:ControllerCV].time for state in states]
E = [state[:Control][:Phi][1] for state in states]
I = [state[:Control][:Current][1] for state in states]

Plot of voltage and current ​

julia
f = Figure(size = (1000, 400))

ax = Axis(f[1, 1],
          title     = "Voltage",
          xlabel    = "Time / s",
          ylabel    = "Voltage / V",
          xlabelsize = 25,
          ylabelsize = 25,
          xticklabelsize = 25,
          yticklabelsize = 25
          )

scatterlines!(ax,
              t,
              E;
              linewidth = 4,
              markersize = 10,
              marker = :cross,
              markercolor = :black,
              label = "Julia"
              )

ax = Axis(f[1, 2],
          title     = "Current",

Plot of SEI length ​

We recover the SEI length from the state output

julia
seilength = [state[:NeAm][:SEIlength][end] for state in states]

f = Figure(size = (1000, 400))

ax = Axis(f[1, 1],
          title     = "Length",
          xlabel    = "Time / s",
          ylabel    = "Length / m",
          xlabelsize = 25,
          ylabelsize = 25,
          xticklabelsize = 25,
          yticklabelsize = 25
          )

scatterlines!(ax,
              t,
              seilength;
              linewidth = 4,
              markersize = 10,
              marker = :cross,
              markercolor = :black)

ax = Axis(f[2, 1],
          title     = "Length",
          xlabel    = "Time / s",
          ylabel    = "Voltage / V",

scatterlines!(ax,
