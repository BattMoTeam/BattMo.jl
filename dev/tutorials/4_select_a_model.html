Selecting a model ‚Äã

As mentioned in the first tutorial, a model can be thought as a mathematical implementation of the electrochemical and transport phenomena occuring in a real battery cell. The implementation consist of a system of partial differential equations and their corresponding parameters, constants, boundary conditions and assumptions.

The default Lithium-Ion Battery Model corresponds to a basic P2D model, where neither current collectors, degradation nor thermal effects are considered. BattMo has implemented several variants of the Lithium-Ion Battery Model, which can be accessed by configuring the model object. In this tutorial, we‚Äôll configure a P2D model with degradation driven by SEI (Solid Electrolyte Interphase) growth.

Load BattMo and Model Settings ‚Äã

julia
using BattMo, GLMakie

Let‚Äôs begin by loading the default model settings for a P2D simulation. This will return a ModelSettings object:

julia
model_settings = load_model_settings(; from_default_set = "P2D")

We can inspect all current settings with:

julia

Since we want to observe SEI-driven degradation effects, we‚Äôll specify which SEI model we'd like to use, and with that enable the use of the SEI model during the simulation.
